## 디자인 원칙

### 1. 따뜻하고 안전한 느낌 (Warm & Safe)

관계 문제를 다루는 서비스이므로, 사용자가 **안전하고 편안하게** 느낄 수 있는 분위기

- 부드러운 그라데이션
- 둥근 모서리
- 따뜻한 색상 톤

### 2. 전문적이지만 접근성 있게 (Professional yet Approachable)

심리학 이론 기반이지만 **어렵지 않게** 전달

- 아이콘과 이모지 활용
- 시각적 다이어그램
- 단계별 설명

### 3. 집중할 수 있는 UI (Focused UI)

대화에 집중할 수 있도록 **미니멀한 인터페이스**

- 불필요한 요소 제거
- 넓은 여백
- 명확한 계층구조

### 트랜지션

| 요소 | 애니메이션 | 시간 |
|------|------------|------|
| 페이지 전환 | Fade | 200ms |
| 카드 호버 | Scale(1.02) | 150ms |
| 메시지 등장 | Slide Up + Fade | 300ms |
| 모달 | Scale + Fade | 200ms |

### 마이크로 인터랙션

- 버튼 클릭: Scale(0.98) → Scale(1)
- 복사 성공: Checkmark 아이콘 표시
- 에러: Shake 애니메이션
